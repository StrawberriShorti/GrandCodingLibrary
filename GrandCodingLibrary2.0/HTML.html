Link Types


A Normal Link: <a href="your website or directory link here">website or directory name</a>

An Image Link: <img src="image link here">

You can also use directory links such as /images/exampleImageOne.jpg

Formats


<b> - Bold text
<strong> - Important text
<i> - Italic text
<em> - Emphasized text
<mark> - Marked text
<small> - Smaller text
<del> - Deleted text
<ins> - Inserted text
<sub> - Subscript text
<sup> - Superscript text

Colors


How to add a preset color to anything:

<h1 style="color:cornsilk;">Example Text<h1>



Borders and coloring them:

<p style="border:2px solid Violet;">



Background Colors

<p style="background-color:rgb(255, 9, 7);">Example</p>

ID Attributes


In HTML you are able to specify the id of an object such as a header or paragraph, this lets you call them later on if needed.

an example:



<h1 id="IDOfAHeader">My Really Cool Header</h1>

To simply navigate to it later on, you can call to it with:

<a> href="#IDOfAHeader" class="nav-link">My Awesome Header Example</a>
