Why Java?

Java is one of the most adaptable coding languages currently available to us programmers.

It can be used for Mobile applications, desktop applications, web applications,
web servers, video games, databases, and more.

The Main Method:

The main method is used to actually run the code you write.
For example if you make a 'car' class, you can use the main method
to create actual objects of car class.

public class Main {
public static void main(String[] args) {
System.out.println("Hello World");
} }

How to Make a Comment

There are two types of comments in Java.
A single line comment starts with //
and a multi line comment begins with /** and ends with **/

// This is a single line comment
/*And this is a multi
line comment*/
